Задача: Чтение из файла 2

В файле filename в каждой строке написаны две координаты точки, разделенные пробелом. Прочитайте файл в массив точек. Решите задачу в один LINQ-запрос. Постарайтесь не использовать функцию преобразования строки в число более одного раза.

Пример файла

1 -2
-3 4
0 2

Написанная вами функция в дальнейшем может быть использована, например, вот так:

Краткая справка

  • IEnumerable<R> Select(this IEnumerable<T> items, Func<T, R> map)
  • List<T> ToList(this IEnumerable<T> items)

Допишите решение в окне ниже:

Подсказка 1

string.Split — разбивает строку на части по разделителю

Подсказка 2

int.Parse преобразует строку в целое число.

Подсказка 3

Каждую строку нужно преобразовать в точку. Преобразование — это дело для метода Select.
